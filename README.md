# titanic_survivors_predictor

This model is made to predict the survival of the passengers who were travelling in titanic.

### Tools used
* Pandas
* Scikit-learn

### Data Sets Used
* **titanic_train:** Used to train the model on the basis of age, sex, class, sibsp, parch, fare, deck, embark
* **titanic_test:** Used to predict the survival of passengers

### Model Details
* **Algorithm used:** Logistic Regression
* **Accuracy:** 82%
