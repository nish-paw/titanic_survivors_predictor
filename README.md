# titanic_survivors_predictor
This model is made to predict the survival of the passengers who were travelling in titanic <br>
Tools used- Pandas,Scilit<br>
it uses two data sets: 
titanic_train - to train the model on basis of age,sex,class,sib.,par.,fare,deck,embarck
titanic_test - to predict the survival of passengers<br>
Algorithm used- LogisticRegression<br>
Accuracy=82%
